## 🔍 Hardest part
Designing the interactive custom **DSA Simulator** (recreating animated sorting algorithms and visual stack states using pure CSS and vanilla JS) and fine-tuning the **Hero Placement Slider** so the background images occupy the entire visual card, adjusting absolute coordinates and negative margins dynamically on desktop without causing layouts to overlap or overflow on mobile breakpoints.

## 🛠 Stack
HTML5 · CSS3 (Vanilla CSS with CSS Variables, Flexbox, and Grid) · Vanilla ES6 JavaScript · **Antigravity Coding Assistant** for tricky layout debugging & assets creation · GitHub Pages

## 📚 What I learned
- **Advanced Responsive Layouts:** Gained a deep understanding of combining Flexbox and CSS Grid layouts to align side-by-side elements on desktop that collapse and stack cleanly on mobile viewports.
- **Stateful JS Logic & Visualizers:** Learned how to translate algorithm logic (like Bubble Sort) into visual dom element changes (swapping heights, shifting classes, and updating color indications dynamically in real time).
- **Interactive UI Components:** Built custom, state-tracked components like autoplay carousels with pause-on-hover mechanics, tab systems, accordion menus, and multi-step registration forms.
- **Cache-Busting Techniques:** Learned how to implement cache-busting query strings on stylesheets and scripts to force immediate browser updates during deployment.
- **Glassmorphism & Neon Design:** Mastered CSS styling tricks using `backdrop-filter`, semi-transparent border borders, overlay linear gradients, and glow shadows.
